# Python CLI Calculator

A simple command-line calculator built with Python.

This project supports basic arithmetic operations, continuous calculations using the previous result, and operation history tracking. It was designed as a beginner-friendly Python project with a clean and modular structure.

## Features

- Addition, subtraction, multiplication, and division
- Continuous calculations using the latest result
- Operation history tracking
- Clear current calculation
- Reset calculator history
- Input validation for numbers and operations
- Safe handling for division by zero

Technologies Used:
Python 3

How to Run:
Make sure Python 3 is installed
Clone the repository or download the files
Open a terminal in the project folder
Run;
python main.py

Available Commands:
C → Clear current calculation
R → Reset calculator history
P → Print previous operations
Q → Quit the program

Supported Operations:
+ or add
- or sub
* or mult
/ or div

Example:
=== Python CLI Calculator ===

Enter first number: 10
Select operation (+, -, *, /, add, sub, mult, div): +
Enter second number: 5
Result: 15

Select operation (+, -, *, /, add, sub, mult, div): *
Enter second number: 2
Result: 30
